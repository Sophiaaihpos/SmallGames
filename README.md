# HTML5 Game Collection

A collection of three fun HTML5 games built with **PixiJS**, featuring Snake, a Color Matching game, and Flappy Bird.

### 1. **Snake**
Control the snake with arrow keys, eat apples to grow and score points!
- **Features**: Two difficulty levels (Easy & Normal), increasing speed in Normal mode
- **Controls**: Arrow Keys to move
- **Goal**: Eat as many apples as possible without hitting walls or yourself

### 2. **Click Game (Color Match)**
Test your attention and reflexes by clicking on words that match their color.
- **Features**: 60-second time limit, progressive difficulty
- **Controls**: Click on matching color/text pairs
- **Goal**: Identify and click correct matches before time runs out. One mistake = Game Over!

### 3. **Flappy Bird**
Navigate a bird through randomly generated pipe obstacles.
- **Features**: Progressive difficulty (pipes get narrower as score increases)
- **Controls**: Press SPACE to make the bird jump
- **Goal**: Fly through as many pipes as possible without hitting them
